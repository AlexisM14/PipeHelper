gestion_BDD_geometries module¶

File: gestion_BDD_geometries.py Author: Alexis Markiewicz Date: 2024-07-08 Description: Ce script permet de définir des fonctions qui permettent de gérer la base de données “BDD_geometries.xlsx”

gestion_BDD_geometries.recuperer_attribut_geo(geometrie, colonne)¶

Cette fonction récupère une valeur pour la géométrie ‘geometrie’ dans la colonne ‘colonne’.

Parameters:

  • geometrie (str) – Le nom de la géométrie

  • colonne (str) – Le nom de la colonne ou chercher

Returns:

La valeur dans la colonne pour la géométrie

Return type:

float

gestion_BDD_geometries.recuperer_coeff_perte_charge_singuliere(geometrie, angle, diametre_entree, rayon_courbure)¶

Cette fonction renvoie le coefficient de perte de charge singulière pour la géométrie, l’angle, le diamètre et le rayon de courbure donné.

Parameters:

  • geometrie (str) – Le nom de la géométrie

  • angle (str) – L’angle de la géométrie

  • diametre_entree (float) – Le diamètre de la géométrie

  • rayon_courbure (float) – Le rayon de courbure de la géométrie

Returns:

Le coefficient de perte de charge singulière

Return type:

float

gestion_BDD_geometries.recuperer_fourchette_geo(geometrie, colonne, nombre)¶

Cette fonction renvoie le nombre juste avant et juste après ‘nombre’, dans la colonne ‘colonne’, pour ‘fluide’ dans la base de données.

Parameters:

  • geometrie (str) – Le nom de la géométrie

  • colonne (str) – Le nom de la colonne ou chercher

  • nombre (float) – Le nombre qu’on souhaite encadrer dans la colonne ‘colonne’

Returns:

Le plus grand nombre, plus petit que nombre dans la liste

Return type:

float

Returns:

Le plus petit nombre, plus grand que nombre dans la liste

Return type:

float
